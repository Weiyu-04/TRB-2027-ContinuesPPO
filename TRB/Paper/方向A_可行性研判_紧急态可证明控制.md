# 方向 A 可行性研判 — 把 gap#1 从「检测器」做成「可证明的紧急控制器」

> **状态：草稿·我的判断·待独立复审**（作者=可达性背景·请把每条"可行性判断"当靶子打）。
> **目的**：判断"紧急态（ρ5·占冲突步 55-60%·当前=会撞的经验黑盒 Alg.1）的真可证明处理"能不能证出来——这是把项目从"工程盾 + 一条窄引理"抬成"原创贡献"的唯一硬路（见 `03` L164 审计 + 本轮 novelty 复审）。
> **红线继承**（CLAUDE §0 + 三份理论稿）：绝不写 provably collision-free / 完整可证明覆盖 / 裸 provably-minimal。本文只研判"能证到什么档"，不制造新 over-claim。

## 0 · 定位：要闭的两个鸿沟

现状（`Paper/可证明层_正式命题_0707.md` 命题3 = gap#1）：
- gap#1 是 **sound 的「迫近不可避」检测器**：`∃t*: R_box(t*) ⊆ O(t*)⊕disk(r_insc)` ⟹ 碰撞不可避。**它只回答"什么时候躲不掉"，不开船避碰。**
- 两个已知鸿沟（`03` L135/L139/L164 自曝）：
  1. **「可避」侧不 sound**：命题3 的补集（gap#1 不触发）**不等于**"可避"（过近似集不被完全包住 ≠ 存在清障轨迹）。旧"可避"判据是 1D 标量（只查转向、漏加减速逃逸）→ 已被判 unsound（L139：真碰撞态 full-search 45% 其实可避·head-on 靠加速/crossing 靠减速）。
  2. **「存在 → 执行」鸿沟**：即便证出"存在安全脱身机动"，实际开船的是 RL + 单步投影**另一条路**（`03` L135：脱身机动 execution-期 deferred·从不被驾）→ 证了存在、没证驾着它走。

**方向 A = 同时闭这两个鸿沟**，做成"可避时可证明避碰（且真执行）+ 不可避时可证明控制严重度"的**紧急恢复层**。

- 他船：恒速 CV 外推 `O(t)=`矩形（中心 `p_m+v_m t`）。ego 内切盘 `disk(r_insc=12.7)`。

## 2 · 命题 A-i：可避时可证明避碰（构造式 + 执行）

**要证的陈述（拟）**：设单个 CV 他船、ego 处于紧急态。给定一个**有限的候选脱身机动族** `M={m_1,…,m_K}`（每个 m_k = 一段具体开环控制律，如"满减速+保向""满加速+满右转""满减速+满右转"…覆盖 L139 的 head-on-加速 / crossing-减速 两类逃逸）。若
  **∃ m_k ∈ M：ego 在 m_k 驱动下、CV 他船外推下，全程 `t∈[0,T]` 船体最小间隙 > 0**，
则 (a) 碰撞**可避**（构造性证明·有显式清障轨迹）且 (b) 紧急控制器**执行** m_k ⟹ **可证明这一对不碰撞（CV 下）**。

**要证/要做的三块**：
1. **单机动的 sound 清障验证**（`m_k` 全程间隙>0，连续时间、非仅采样点）：ego 在恒 a、恒 ω 下轨迹是已知曲线（弧/近似 clothoid），他船是 CV 直线；两者最小距离需**区间/Lipschitz 上下界**或闭式，做到"采样点之间也不碰"。→ 这基本就是 `maneuver_verified` 要干的事（**已有骨架**）。
2. **执行覆盖**：紧急态让控制器**接管**、开环（或滚动）驾 `m_k`——闭"存在→执行"鸿沟。=系统设计改动（§4）。
3. **🔴 命门：这个"构造式可避证书"够不够紧**——即 `{gap#1 判不可避}` 与 `{∃m_k 清障}` 之间的"**未决区**"（既证不了不可避、也证不了可避）有多大。未决区大 = 刻画不完整 = 贡献弱。

**可行性判断（经独立复审·【下调】）**：
- 1+2 = **低-中**〔原写"中-高"·复审证伪·下调〕。**关键坑（我原漏）**：现有 `maneuver_verified`（`usv_colregs.py:724-737`）是**离散采样 + 他船每边膨胀 350m（dobs,safety=2·l_obs·175m 船→~875m）**的检查器——它的"sound"来自**巨膨胀**，**不是连续时间最小间隙下界**（`03` L134 实测：dt_sim=0.5 的"已验证"机动在细积分 0.1/0.05 下 **4.2% 翻 unsafe**）。所以块1（连续时间 sound）**根本不"已有"**，是没证过的硬部分。
- **🔴🔴 tightness↔soundness 死结（我原完全没看见·这才是 A-i 命门）**：A-i 要"未决区小"（紧），但骨架的 sound 靠 350m 膨胀——**保膨胀 → 几乎没机动能清 875m 膨胀障碍 → 未决区巨大**；**去膨胀求紧 → 立刻暴露采样点之间那 4.2% 未证间隙**。二者不可兼得，不是"接线"能过的。
- **gap#1 的干净几何不能反向复用**：命题3 用 ego **内切盘 under-approx** 证"不可避"（保守方向对）；A-i 证"不碰"要**相反方向**——用 ego **外接圆/全矩形**保最外点清障 → 面对**旋转矩形 vs 运动旋转矩形的连续时间最小距离（非凸）**，要现证。且"满减速"机动 v 饱和到 0、轨迹退化非干净弧（`03` L9/L10 尾巴）。
- 3（未决区）= **决定性未知·且结构性证据指向"大概率偏大（>15%·可能是 ρ5 态多数）"**：gap#1 只在 imminent 末端（t*≤7s）触发、is_emergency 在 733m 就进 ρ5 → 中间一条宽带；`03` L141 已推翻"head-on 加速/crossing 减速"干净规律（逃逸强几何依赖·无规律）→ 有限小机动族必然打补丁留缝。**我原设的"<10-15% → 近乎完整"门槛无任何现有证据支撑=乐观预期。**

**风险/边界**：机动族有限 = 充分非必要（保守）；单障碍；CV（机动他船破保证）；T 有限视界；**+ 连续时间 sound 未证 + 非凸最小距离 + 去膨胀求紧 = 真研究难点，非工程接线**。

## 3 · 命题 A-ii：不可避时可证明控制碰撞严重度

**要证的陈述（拟·拆两档，别混）**：当 gap#1 判不可避时，控制器执行的机动使某"严重度"泛函（如撞击相对速度 / 侵彻深度 / 最近侵入量）满足保证。

- **A-ii-弱「可证明有界」**：控制器在机动族 `M` 里取"严重度最小的那个成员" `m_k*`，则严重度 ≤ 该 `m_k*` 的（可验证）上界。**= 可证明的上界，不是真最小。**
- **A-ii-强「可证明最小」**：严重度 = 全体可行控制上的真最小。

**要证/要做**：
- 弱档：`m_k*` 的严重度上界 = 前向验证（同 §2 块1）→ **可证**。
- 强档：需**最优控制 / reach-avoid 带代价** 的可验证最小——此非线性系统几乎无闭式 → 只能**数值 HJ / 可达集最优**（丢掉干净闭式故事、算力贵、且"可证明最优"要严格数值证明很难）。

**可行性判断（经独立复审）**：
- A-ii-弱（有界）= **中-低**〔原写"中"·下调〕：它"= 前向验证（同 §2 块1）"，**继承块1同款连续时间 soundness 难点**（严重度在碰撞时刻取值·碰撞可能落采样点之间·要 sound 的连续时间上界同是非凸最小距离问题）。"机动族内最优、非真最小"措辞本身诚实、红线没破。
- A-ii-强（最小）= **低**（大概率只能数值 HJ·"可证明最小"在连续非线性下极难严证）。

**⟹ 建议：A-ii 只 claim「可证明有界严重度 / 机动族内最优」，绝不 claim「provably 最小碰撞」**（与 `03` L139/L164 红线一致·别重犯我上轮的 over-claim）。

## 4 · 系统设计改动（必须正视）

闭"存在→执行"鸿沟 ⟹ **紧急态 ρ5 由验证控制器接管、RL 让位**（RL 只主导 ρ0-ρ4；ρ5 交给 A-i/A-ii 的构造机动）。含义：
- **更诚实、更强**：紧急态从"黑盒 Alg.1（会撞·无保证）"换成"验证过的构造机动（CV 下有保证/有界）"——这正是贡献所在。
- **代价**：紧急态不再"纯端到端 RL"（但紧急态本就是 fallback·且 Krasowski 也是切 Alg.1·故不吃亏·反而更可辩护）。
- **🔴 被低估的反噬（复审抓·我原漏）**：`03` L192 I 是同款现成反例——一个几何/验证型接管控制器把船带到离目标 5.7m 但朝向偏 30°>门、无"放弃/交还"判据 → 恶化到 102° → 把**训练分布外状态甩回 RL** → RL 进 ρ5 → 绕圈 → **撞（T-1785）**。同款机理对 A-i：**接管时刻的态由 RL 驾进来，可能已在未决/不可避区**，此时无 certified m_k → 落 best-effort → L192 I 重演。**执行闭合不消灭碰撞，只是把它重定位到"接管发生在未决/不可避区"**。+ 交接瞬态 / `_prev_rho`·EC 边沿跨步态 / 开环机动初值敏感——都要正视。
- **接线**：`As(ρ5)` + `maneuver_verified` 骨架已在 → 接进连续盾 ρ5 分支 + 接管逻辑 + **"放弃/交还"判据**（须走 CLAUDE §2 最高标准 + 对抗审）。**含上面反噬 = 工程量中-高·非"中"。**

## 6 · 多障碍 / 机动他船（诚实边界）

- **多障碍**：A-i 的机动 `m_k` 须**同时**清所有他船 → 机动族在各他船清障集的**交**；交空 → 落 A-ii（有界严重度）。可避性**不合成**（`可证明层_多障碍扩展_0716.md` 命题3-N）→ 多障碍下"可避"更弱。
- **机动他船**：CV 假设破 → A-i/A-ii 保证均失效（写"CV 世界内"）。
- **数据命门**：基准全单他船 CV → 多障碍/机动**经验背书都拿不到**（同 `03` L189 H 老问题）。⟹ 方向 A 的经验演示也只在单障碍 CV。

## 7 · 总体可行性裁决 + 诚实定位

**我的判断（经独立复审·【下调】）**：
- **能证出来的**：A-i 的**块2（执行接管接线）= 中**；A-i **块1（连续时间 sound + 去膨胀求紧）= 低-中·且是决定性硬部分**（非凸旋转矩形最小距离·gap#1 几何不能反向搬）。A-i 作为"强贡献"整体把握 = **低-中，不是我原写的"中-高"**。A-ii-弱（有界）= 中-低（继承块1难点）。
- **证不出/只数值的（低·别押）**：A-ii-强（可证明最小碰撞）；紧的必要"可避"条件（联合可达集）。
- **决定性未知·且结构证据指向偏大**：§2 块3 未决区**大概率 >15%（可能是 ρ5 态多数）**——gap#1 imminent-only 与有限开环族无规律两头夹。
- **🔴 命门单点**：`§8` 那次**连续时间最小间隙下界（去膨胀·1 个机动）能不能干净证出来**——证得下 A-i 才有底；证不下，A-i 的"紧"就是空的。

**够不够"原创、非 A+B、解真问题"**（经独立复审·收紧）：
- **🚫 绝不能靠"近乎完整刻画"这条**〔原稿这个提法擦了"完整可证明"红线·删除〕：即便未决区小，那也只是**两个保守充分条件（gap#1 判不可避 + ∃m_k 判可避）+ 经验小缝**的**拼装**，**不构成可判定的必要充分边界**；形式化审稿人会把"近乎完整刻画"读成"可避/不可避边界可判定"（=项目 L138/L141 明确放弃、列 future-work 的东西）。
- **能站的原创框法（唯一）**：**"首个针对连续 COLREGs 结构化紧急态、把'检测'推进到 sound + 可执行的恢复层"**（Krasowski 紧急=黑盒·CBF 类回避 COLREGs 结构化紧急）。且净新数学 = "针对此动力学的 sound 可避证书 + 执行闭合"，**ICS/可达集概念不新（Fraichard 2004 / Althoff）·写作须诚实、非概念首创**。
- **若块1证不出 / 未决区大** → 退回"gap#1 是引理（但它本身已 sound、已过审、是真种子）"，则转方向 B（多船·被数据卡）或接受扎实 demonstration 论文。**早知道 > deadline 才知道。**

## 8 · 先做哪个零算力验证（决定投不投·最便宜的 go/no-go）

**决定性先验（纯数值·不烧训练卡·顺序已按复审改）**：

**① 🔴 make-or-break（必须先做·原稿标"可选"是错的）= 单机动连续时间 sound 下界**：挑 1 个 m_k（如满减速+保向），**去掉 350m 膨胀**，真做一次**连续时间最小间隙下界**（区间算术/Lipschitz·处理旋转矩形 vs 运动矩形非凸最小距离 + v 步内超调 + 满减速 v 饱和退化）。
- **证得下、且不非凸炸开 → A-i 有底**，继续 ②。
- **证不下 → A-i 的"紧"是空的**（保膨胀则未决区巨大、去膨胀则不 sound）→ 方向 A 退回"gap#1 是引理 + demonstration"或转 B。**这一步单独就能否掉方向 A，最该先做。**

**② 三分区实验（须用 ① 的去膨胀+连续时间 sound 判据·别用现成 maneuver_verified）**：采样 ρ5 态，统计 **{gap#1 判不可避} / {∃m_k 连续时间 sound 清障=可避} / {两者都不=未决区}**。
- ⚠️ **别用现成 `maneuver_verified` 跑这个**——它是离散采样 + 350m 膨胀，会把"不可避"和"350m 膨胀下不可避"混算（膨胀不去→未决虚高；连续时间 sound 不补→可避虚高）=**数字会骗人**。
- 未决区偏小 → 值得投；偏大（结构证据预期 >15%）→ 机动族太保守 → 扩族（更难证）或重估。**绝不写"近乎完整刻画"**（擦红线）。

**③ 机动族设计小扫**：先用 `03` L141 的逃逸机理（注意 L141 已判"无干净规律·强几何依赖"）设 4-8 个基本机动看覆盖；不够再加复合（复合机动的连续时间 sound 更难）。

**产出 = 一张"方向 A 到底能不能成"的证据表**（配你的可达性判断）拍板：投 A / 转 B / 收 demonstration。**核心：① 是命门单点，先证它。**
