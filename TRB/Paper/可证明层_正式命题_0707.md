# TRB 2027 · 可证明层正式命题（formal propositions）· 2026-07-07

> 论文可证明层的正式数学陈述 + 假设清单 + 证明。**诚实分档**：只写能严证的，证不出的标经验/存在性级。
> 依据 `03` L133-L141（原推导）、L164（诚实审计）、L165（gap#1 攻坚）。红线：**绝不写 provably collision-free / provably compliant / complete provable coverage**。

## 0 · 系统设置（notation）

- **本船动力学**（忠实 Krasowski `vessel_dynamics_yp`，SR108 集装箱船）：
  ẋ = v·cosθ, ẏ = v·sinθ, θ̇ = ω, v̇ = a，控制约束 |a| ≤ a_max=0.24 m/s², |ω| ≤ ω_max=0.03 rad/s，速度 0 ≤ v ≤ v_max=9.5 m/s（env 每步 clip；步内可短暂达 v_max+a_max·Δt=11.9），决策步长 Δt=10 s。船体矩形 175×25.4 m：外接圆半径 R_circ=½√(175²+25.4²)=88.4 m，内切圆半径 r_insc=½·25.4=12.7 m。
- **他船模型**：占据用船体矩形（或外接圆保守）；除非特别声明，规则态势（stand-on）下取**恒速（CV）**外推。
- **符号**：p_e,p_m 本/他船位置；v⃗_e=v_e(cosθ_e,sinθ_e) 本船速度矢量；ĥ=(cosθ_e,sinθ_e) 艏向单位向量，ĥ⊥=(−sinθ_e,cosθ_e) 其法向。

## 命题 1（远场 Stage-0 · 单步无碰）— 状态：**PROVEN（严格·初等引理）**

**陈述.** 设单个他船与本船中心距 d = ‖p_e−p_m‖。若
  **d ≥ d_safe + v_obs,max·Δt + ρ_ego**
则一个决策步后两船体（外接圆保守占据）必不相交，即下一步不可能发生碰撞。其中 d_safe = R_circ,ego + R_circ,obs，ρ_ego = v_max·Δt + ½·a_max·Δt²（本船一步最大位移）。

代入常数：R_circ,ego = ½√(175²+25.4²) = 88.4 m（本船宽已知 25.4）；**R_circ,obs = ½√2·l + dobs,safety = ½√2·175 + 2·175 = 123.74 + 350 = 473.74 m**（他船宽不在状态 → 取方形保守上界 w=l=175 算外接圆，再加 Table II 的 dobs,safety=2·l 安全裕度）；故 d_safe = 88.4 + 473.74 = 562.16 m，ρ_ego = 9.5·10 + ½·0.24·100 = 107.0 m，v_obs,max·Δt = 9.5·10 = 95 m ⟹ **阈值 = 764.16 m**（工程取 ~780 m 留裕度）。

**假设.**
(A1) 单他船（多他船须对每个 i 分别满足 d_i ≥ 阈值）。
(A2) **他船速度 v_obs ≤ 9.5 m/s** —— Krasowski 只对本船设 v_max=9.5；他船此界须显式假设。**已验（全池·`03` L167）：HandcraftedTwoVessel 基准 2000/2000 场景、342000 障碍态，max = 7.10 m/s（=7.0989·场景 T-972），0 超 9.5（裕度 ~2.4 m/s）**（初 n=60 max 7.07 → 全池复验 max 7.10·0 超；真实 AIS 数据速度更高则阈值须抬到 d_safe + v_obs,max·Δt + 107）。
(A3) 本船 v ≤ v_max（env clip 保证步末）。
(A4) 他船一步内近似恒速（stand-on）。
(A5) 外接圆保守占据（圆盘不交 ⟹ 船体不交）。

**证明.** 一步后两船中心距 d_next = ‖(p_e+Δp_e) − (p_m+Δp_m)‖ ≥ d − ‖Δp_e‖ − ‖Δp_m‖（三角不等式）≥ d − ρ_ego − v_obs,max·Δt。由假设 d ≥ d_safe + v_obs,max·Δt + ρ_ego，得 d_next ≥ d_safe = R_circ,ego + R_circ,obs，即两外接圆不相交，故两船体不相交。∎

**注.** 三角不等式与方向无关 ⟹ head-on 即真最坏情形，无几何缺口。这是全可证明层**最 sound 的一块**（初等引理）。是单步充分条件（保守），非不变性、非近场保证。当前**未在代码实现早退**（可加 O(1) 远场 fast-path）。

## 命题 2（每步让路方向合规）— 状态：**by-construction PROVEN（作用域受限）**

**陈述.** 在被状态机分类为让路态势（ρ∈{head_on, crossing, overtake}）且投影二次规划可行（needs_fallback=False）的每一决策步，实际执行的安全动作 u_safe 落在 COLREGs 规定的合规方向半平面内（如让路右转 ω ≤ −ω_turn）。

**假设.** (B1) 单他船；(B2) 状态机 ρ 分类正确（继承 ~12.6% cp-gap + Lemma-1 tie-break 洞）；(B3) QP 可行（非兜底）；(B4) 作用域 = 让路态势的相遇。**实测 give-way 触发率 ~13–16%（encounter/episode 级：X/933 局有让路触发·非 per-step；stand-on 违规才按步计）**，两次手算 933 场景；overtake=0。

**证明.** 合规半平面作为硬区间约束进 QP（w_hi = −ω_turn 等），投影解必满足之。∎（by-construction，无需独立定理）

**🚫 作用域红线.** **不覆盖紧急态 ρ5（占冲突步 55–60%，is_emergency 结构性旁路合规约束）**及 relaxed/collision_min 兜底（丢半平面）。故只能写 **"每让路步方向合规（provably give-way-direction-compliant, per give-way step）"**，**绝不写"每步都合规 / provably compliant"**（实测违规 1.2–1.6/局非零）。

## 命题 3（迫近不可避证书 · gap#1）— 状态：**SOUND 充分条件（对抗审 0 假阳/~2.9万场景 + 主窗口硬化成真证明）**

**陈述.** 单个恒速（CV, stand-on）他船。定义 t 时刻本船**可达中心过近似集** R_box(t)（艏向系矩形，中心取恒速外推 c_e(t)=p_e+v⃗_e·t）：
  纵向半轴 B_lon(t) = ½·a_eff·t²，  横向半轴 L_lat(t)；
  a_eff = √(a_max² + (v_bnd·ω_max)²)，v_bnd = v_max + a_max·Δt = 11.9 m/s；
  L_lat(t) = (v_bnd/ω_max)(1−cos ω_max t)  若 ω_max·t ≤ π/2，否则 (v_bnd/ω_max) + v_bnd·(t − π/(2ω_max))。
设他船船体矩形 O(t)（中心 p_m+v⃗_m·t，朝向 θ_m）。若
  **∃ t* ∈ [0,T]: R_box(t*) ⊆ ( O(t*) ⊕ disk(r_insc,ego=12.7 m) )**
则碰撞**不可避**（任一可行控制序列都在 t* 撞上他船）。

**假设.**
(A1) 单他船，**per-pair**（per-pair 不可避 ⟹ 全局不可避；per-pair 可避**不可**合成）。
(A2) 他船**恒速 stand-on**（机动他船则 O(t) 变，保证失效——写"CV 世界内不可避"）。
(A3) 本船 v ≤ v_bnd=11.9 步内（已含 env clip 的步内超调）。
(A4) 本船内切圆 r_insc=12.7 ⊆ 船体（任朝向；12.7=½ 船宽，长边零余量须复核）。
(A5) 判据仅在触发区 ω_max·t* ≤ π/2（≤52.4 s）内成立——**主窗口亲核**：触发受纵向二次项限制在 t_crit_lon=√(2(½·175+12.7)/a_eff)=**21.6 s < 52.4 s** ✓（且硬化后 L_lat 用全 t 包络，本假设自动满足）。
(A6) **单时刻 containment**（充分非必要）。
(A7) **占据 O(t) under-approx：obs_width ≤ 真实他船宽、L_obs ≤ 真实他船长**（调用方保证的 soundness 前置）。传大于真实尺寸 → O 过近似 → 可能假阳（unsound·对抗审实证 obs_width=60 对真宽 25.4 会假阳）；基准两船皆 SR108（25.4×175）→ 默认精确，真实/更窄他船须显式传真宽。

**证明.**（三步）
1. **R_box 过近似真可达中心.** 本船速度矢量变化率 ‖dv⃗/dt‖ = ‖a·ĥ + v·ω·ĥ⊥‖ = √(a²+(vω)²) ≤ √(a_max²+(v_bnd·ω_max)²) = a_eff（因 |a|≤a_max, v≤v_bnd, |ω|≤ω_max）。由有界加速度双重积分，位置对恒速外推的偏移各向 ≤ ½·a_eff·t²（纵向界）。横向偏移 |y′(t)| = |∫₀ᵗ v sin(Δθ)ds| ≤ ∫₀ᵗ v_bnd·sin(ω_max s)ds = (v_bnd/ω_max)(1−cos ω_max t)（ω_max s ≤ π/2 内 |sin Δθ|≤sin ω_max s；超之用转-90°-后直行包络）= L_lat(t)。故真可达中心 ⊆ R_box(t)。
2. **内切盘 under-approx 船体.** disk(12.7) 以本船体中心、半径=½ 船宽 ⊆ 船体（任朝向）。
3. **单时刻困住 ⟹ 撞.** 若 R_box(t*) ⊆ O(t*)⊕disk(12.7)，则任一可达本船中心距 O(t*) ≤ 12.7 ⟹ 本船内切盘(⊆船体)与 O(t*) 相交 ⟹ 本船体与他船体相交 = 碰撞。因对**所有**可达位置成立，任一轨迹在 t* 撞。∎

**验证（`03` L165）.** 6-agent 对抗审（含 20000 随机 + 735 curated·triple-switch bang-bang 逃逸搜·精确船体碰撞）**0 真假阳**；主窗口硬化（v_bnd/L_lat 包络）后 battery+1500 fuzz 0 假阳、仍非空（触发 t*≤7 s）。

**🚫 诚实覆盖（必写论文·别夸）.**
- ✅ **迫近不可避 = sound**（保守单侧充分条件，替换旧 unsound turn-only 版）。
- ❌ **早预警**（提前很多步断定）：NO——末端 classifier 非规划 cost。
- ❌ **完备/必要性**：NO——no-fire ≠ 可避（单时刻 containment）。
- ❌ **多障碍**：NO（per-pair 可避不合成）。
- ❌ **机动他船**：NO（须 CV）。
- **绝不写** provably-zero-collision（这只是紧急层 classifier，非盾整体保证）。

**能写的句子.** "紧急层采用 sound 充分条件判迫近不可避：单恒速他船下，若某 t*（ω_max·t*≤π/2）本船可达中心过近似 ⊆ 膨胀他船体，则任可行控制皆撞；保守单侧，~2.9万对抗场景 0 假阳；替换此前只查转向、会漏加减速逃逸的 unsound 判据。"
