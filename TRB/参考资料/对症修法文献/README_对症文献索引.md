## A. 奖励塑形 / PBRS（对症修法地基）
- **Ng1999_PBRS地基_策略不变性.pdf**（Ng, Harada, Russell 1999, ICML）= PBRS 理论地基：F=γΦ(s′)−Φ(s) 不改最优策略集。我们 Φ_A + Φ_xtrack 两项之和保策略不变性的依据（=盾可证明合规不被污染的一部分）。
- **Trott2019_自平衡塑形_距离奖励局部最优_1911.01417.pdf**（NeurIPS 2019）= 经典出处：naive 距离塑形致"停在目标附近/局部最优"。PPO 兼容。对症问题的命名来源（但我们实测根因是终端控制缺陷·非吸引子·见 L88）。
- **DeLellis2024_控制需求奖励塑形_capture停留保证_2311.10026.pdf**（IEEE TCST 2024）= 最对症：把"settling time 内进入目标区 G + permanence 内停留"编进奖励 + 可证明保证。与我们"朝向+时间门控目标区·须真进入"几乎 1:1。capture/permanence 思想来源。
- **MullerKudenko2025_PBRS有效性理论_2502.01307.pdf**（AAMAS 2025）= 理论警示：单连续势难根治 stop-at-attractor·提示用非对称/线性势或把 capture 做成终端真实奖励。支持我们【线性 prox_lat 替高斯】的选择。

## B. 安全盾/动作投影对性能·终端的副作用（创新点角度 + 诚实护栏）
- **Pizarro2024_训练期安全滤波_盾伤性能_2410.11671.pdf**（IEEE RA-L 2025）= "盾因控制器与滤波器分离而损性能"主引文 + 训练期纳入盾 + 惩罚修正量（≈我们 aliasing 惩罚）。⚠️ 压低"发现盾副作用"新颖性。
- **Oh2024_可证明最优_宽松盾不损性能_2510.18082.pdf**（2025）= 证明"足够宽松(最小干预)的盾不损渐近最优"。⚠️【创新点入口】：若 COLREGs 方向约束使终端允许集"不够宽松"→ 这是"为何终端失速"的机制框架（permissiveness）。
- **Grover2022_多机器人deadlock_盾致停滞_2206.01781.pdf**（IJRR 2023）= "终端失速/停在目标前"的控制理论专名 = **deadlock**（安全项 vs 目标项对冲）。给我们现象权威术语。
- **Vaaler2024**（在 `../竞争文献/Vaaler2024_..._2312.01855.pdf`）= 海事+连续+盾·结论【相反】(PSF 不损性能)。**诚实护栏**：绝不能把"盾损终端"当公理。

## 用法（对症修法 Φ_xtrack）
- 数学保证：Ng1999（PBRS 不变性）。
- 形式选择：MullerKudenko2025（线性优于高斯）+ DeLellis2024（capture 思想）。
- 写作 framing：deadlock（Grover2022）命名现象；Pizarro/Oh 定位盾副作用（诚实标已知）；Vaaler 当反方护栏。
- ⚠️ 主贡献仍是盾；对症修法是"让连续动作追平离散"的二级方法·份量随 A/B 结果定·不预先吹。
