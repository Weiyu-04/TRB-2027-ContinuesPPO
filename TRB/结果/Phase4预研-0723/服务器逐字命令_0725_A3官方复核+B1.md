## 实验 1 · A3 官方 --gates 复核（只跑让路态 860·别跑全 128985）
> ⚠️ `--gates` 会对 IN_JSONL 里【每一个】态跑机动族搜索（89 条×官方 odeint）。全 128985 态会跑到天荒地老。
> **只需让路态 ρ2/3/4（860 个）**——先过滤出来，再跑。

### 1a · 过滤出让路态（秒级）
```bash
/root/miniconda3/bin/python -c "
import json
inp='/root/trb/结果-A3-0723/a3_giveway_states_BIG.jsonl'
out='/root/trb/结果-A3-0723/a3_giveway_ONLY.jsonl'
n=0
with open(out,'w') as fo:
    for l in open(inp):
        if json.loads(l)['rho'] in (2,3,4): fo.write(l); n+=1
print('give-way states written:', n)   # 须=860 (head-on63/crossing767/overtake30)
"
```

### 1b · 冒烟（先跑 50 个·确认无 Traceback + 表能打印 + 估时·别直接上全量）
```bash
head -50 /root/trb/结果-A3-0723/a3_giveway_ONLY.jsonl > /root/trb/结果-A3-0723/a3_gw_smoke50.jsonl
PYTHONPATH=/root/trb/代码 IN_JSONL=/root/trb/结果-A3-0723/a3_gw_smoke50.jsonl \
  /root/miniconda3/bin/python -B /root/trb/代码/m1_dock_wip/a3_giveway_probe.py --gates 2>&1 | tail -30
```
**冒烟验收**：末尾打出「A3 门报告 A」表（含"门2全体%"和"门2in-A%"两列）·无 Traceback。记下 50 态用时 → ×17 约等全 860 用时（预计几十分钟~1-2 小时·官方 odeint 比本机 RK4 慢）。冒烟绿了再跑全量。

### 1c · 全量（screen 后台·860 让路态）
